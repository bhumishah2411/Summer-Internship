# 📌 Project

## AI-Powered Automated Recruitment System (ATS)

An enterprise-grade **Applicant Tracking System (ATS)** designed to automate and streamline the complete recruitment lifecycle, from manpower request creation to candidate onboarding.

The platform provides role-based dashboards for **Department Heads (HODs), HR Recruiters, and HR Manager**, enabling efficient recruitment management, candidate tracking, approval workflows, and hiring analytics.

---

# ✨ Key Features

## 📝 Manpower Request Management (MRF)

- Create and submit digital Manpower Request Forms (MRFs)
- Multi-section recruitment request workflow
- Position details and hiring requirements
- Replacement/New position management
- Qualification and eligibility criteria

---

## ✅ Approval Workflow

- Admin approval and rejection workflow
- Paper-style requisition preview
- Review comments for rejected requests
- Automated request status management

---

## 👨‍💼 Candidate Management

- Resume upload and storage
- Candidate profile management
- Experience tracking
- Resume attachment management

---

## 📊 Recruitment Dashboard

- Role-based dashboards
- Recruitment analytics
- Job opening management
- Candidate pipeline monitoring

---

## 🔄 Applicant Tracking Pipeline

Manage candidates through different hiring stages:

- New
- Screening
- Shortlisted
- Interview Scheduled
- Offered
- Joined
- Rejected

---

## 🎯 Automatic Vacancy Fulfillment

- Automatic job closure after required hires
- Date of Joining (DOJ) management
- Vacancy fulfillment tracking
- Filled position management
